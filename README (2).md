# Finance Dashboard

This is a simple finance dashboard built using React. It helps users track their income, expenses and overall balance in a clean and simple way.

## Features

* Shows total balance, income and expenses
* Displays transactions in a table
* Filter transactions by income and expense
* Role based UI (Viewer and Admin)
* Pie chart for spending breakdown
* Insights like highest spending category

## Tech Used

* React
* Tailwind CSS
* Recharts

## How to run

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

## Notes

I focused on keeping the UI simple and clean. I used mock data and handled all logic on frontend as required. The goal was to make the dashboard easy to understand and use.
